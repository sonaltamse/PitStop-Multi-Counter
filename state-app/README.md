# PITSTOP MULTI-COUNTER

A React-based application themed around Formula 1 (F1) racing, designed to manage multiple pit stop counters. This app allows users to track and adjust counters for various pit stop activities, with customizable increment/decrement steps for each counter.

## Features

- **Multiple Counters**: Manage up to three independent counters simultaneously.
- **Customizable Steps**: Adjust the increment/decrement step for each counter individually.
- **F1 Theme**: Styled with an F1-inspired design, including the official F1 logo.
- **Real-time Updates**: Counters update instantly with user interactions.
- **Responsive Design**: Optimized for various screen sizes.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/state-app.git
   cd state-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to view the app.

## Usage

- Use the input fields to set the step value for each counter.
- Click the "INCREMENT" button to increase the counter value by the specified step.
- Click the "DECREMENT" button to decrease the counter value by the specified step.
- Each counter operates independently, allowing for simultaneous tracking of different pit stop metrics.
